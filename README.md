Source: Meta Ads performance reports (Facebook & Instagram) provided in PDF format.

The dataset includes:
Impressions, clicks, shares, comments, purchases
Campaign-level and ad-type performance
Audience demographics (age, gender, country)
Time-based data (hourly, weekly, monthly)
Budget and spending metrics

Extensive data cleaning was performed to remove unnecessary text such as background labels, links, and metadata, ensuring analysis-ready structured data.
